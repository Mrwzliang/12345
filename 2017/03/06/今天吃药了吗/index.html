今天吃药了吗

发表于 2017-03-06 更新于 2021-06-21

有个人经常问我该不该跟他男朋友分手，我说该！我觉得站在爱情的角度来说，最终走到一起的两个人一定是因为无法割舍的依恋和喜欢，而不是出于合适或现实，那才是我理想中的恋爱或者婚姻应有的模样。

我没有尝过爱情的滋味，这的确是一个很大的遗憾，更让我伤心的是，我也并非一个没有感情的人，相反，我的内心对爱情充满了渴望。仔细分 析来，从小到大，遇见过好许欢喜的女孩子，可是自己也永远停留在这个地步。我也不知道自己到底怎么了，可能被封建教育洗脑了吧，是的，我们不该谈恋爱。也可能自己上辈子是个僧人吧，注定这辈子在爱情方面没有大的成果。 好想这辈子还做一个和尚啊，哪怕是当一个天天敲钟，挑水的和尚也行！那样就不会遇到不该遇到的人，就不会因为感情而悲伤；这么说来，当个和尚真的很幸福！因为爱情她真的是把双刃剑，找不到相互依赖人，那可是真的痛苦，所以你可不要随便喜欢一个人，要是她不喜欢你，你这辈子该怎么办？可是那又怎么样？爱情本身就很珍贵，也很昂贵，不是大部分人都没有享受到吗？落花是有意恋流水，流水也的确无心恋落花。可是不知道为什么心里还是很难受，很不甘心。可是你还不是妥协？
拼死聚不齐的財，努力散不完的情；烟云飘渺无需顾，管他阴雨管他晴。是的，人活着要负责任，你还有你的亲朋好友，记得报答他们，你就必须听他们的安排，你就必须要找一个现实的，合适的对象。结婚也许本来就不一定要互相喜欢。说白了你已经错过了追求恋爱的时间段。你要完全认清自己，盲目给自己定位受伤的总是自己。你喜欢，也就仅仅是你喜欢，至于她，:-(。
一直以为自己给自己的兴趣远远大于别人给的，可是，我错了。我发现现在的自己变了，变得患得患失，这种感觉真的很难受，不管做什么自己有兴趣的事，都不管用了，迎来的只是无穷的悲伤，你说过，男孩子不应该这样。
骗了别人也骗了自己是圣人，骗了别人但骗不了自己是骗子，骗不了别人却骗了自己是精神病。我想我就是神经病:-(。好想有个追求自己爱情的机会，尽管没有经验，没有出色的外貌。也许别人会笑你是个小学生。是的，我还是孩子，很天真。
人活得越久，回忆就越多，这里面亦包括很多不那么美好的回忆，所以对于记忆力比较好的人——譬如我来说，是有点痛苦的。多么希望能一直停留在比较年轻甚至幼稚的阶段，这样脑海里就不会储存太多挥之不去的、想起来会让人揪心或者唏嘘的事物。不过代价是美好的经历也会相对变少，但我宁愿如此，做一个故事少一点的人也好，至少可以一夜无梦，安睡到天明。
多么希望她也很喜欢我啊，希望她相信缘份，相信生死轮回，相信我才是她这辈子要找的人。
一年的光景又过去了，所谓“醒来或者吃饱又是一年，相遇然后分别就在一天”。
